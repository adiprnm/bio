Dulu, kalau mau deploy aplikasi ke server tuh ribet banget.

Harus setup ini-itu:

  • Bahasa pemrograman
  • Database
  • Firewall
  • SSL
  • Redis
  • dan yang lainnya

Belum lagi kalau mau deploy banyak aplikasi dalam satu server, dengan versi bahasa pemrograman yang berbeda-beda. Bukannya fokus ship product, yang ada malah ngurusin hal-hal teknis seharian.

Mau pakai teknologi macam Kubernetes pun rasanya terlalu overkill untuk aplikasi skala kecil.

Sekarang, deploy aplikasi jadi lebih simple dengan bantuan Dokku.

Bayangkan kamu punya platform macam Heroku, tapi ini versi gratisnya.

Kamu cuma perlu setup Docker dan Dokku di server, lalu push changes dari komputer kamu via Git. Boom! Aplikasi kamu langsung ke-deploy ke server.

Udah nggak perlu lagi setup ribet-ribet.
